A Memória Ram ou "Memória de Acesso rápido Aleatório", é o componente que serve de armazém para os dados que o computador está a utilizar no momento, e como o nome diz, o acesso rápido significa que a memória envia informações em uma alta taxa por segundo, a fim de fazer com que o processador analise os dados mais rapidamente.
